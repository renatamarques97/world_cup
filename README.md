# World Cup

## Ruby version
2.6.4

## Rails version
6.0.2.1

## System dependencies
Postgresql

## Configuration
`bundle install` `yarn install`

## Database creation
`rails db:create`

## Database initialization
`rails db:migrate` `rails db:seed`

## How to run the test suite
`bundle exec rspec`

## Run the API server
`rails server`
